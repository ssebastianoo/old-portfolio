How it works

Post something in a server
and whatever you post (a text,
a link or an image) will appear in all the
servers where the bot is!

Commands:

    :post "title" "description" | Make a simple post

    :post "title" "description" "image link" | Make a post with an image

    :setup | Setup the bot in the actual servee (only admins can do this command)
