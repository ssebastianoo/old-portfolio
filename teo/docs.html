Moderation Commands

Member is a user in the actual guild.
User is user in a non-specific guild.

 ban [member] [reason]

  • Ban a member

    •  idban [user id] [reason]

  • Ban a member that is not in the actual guild, giving the user id

    •  kick [member] [reason]

  • Kick a member

    •  mute [member] [reason]

  • Mute a member, it will automatically set to false permissions to write in all channels

    •  unmute [member]

  • Unmute a member

    •  clear [number of messages]

  • Delete a number of messages in a channel
